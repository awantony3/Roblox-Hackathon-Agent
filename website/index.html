01

Installation

From ZIP to running agent.

Complete these steps in order. Every command can be copied with one click.

Before you begin
  • ✓ macOS computer
  • ✓ Node.js 18+
  • ✓ Roblox Studio
  • ✓ OpenAI API key
  1. 01

    Download and extract

    Download the latest main branch, open the ZIP, and move the extracted folder somewhere easy to find.

    Download main.zip ↓
  2. 02

    Open the project in Terminal

    Type cd, add a space, drag the extracted project folder into Terminal, and press Return.

    cd /path/to/Roblox-Hackathon-Agent-main
  3. 03

    Create the environment file

    Copy the safe example configuration, then open the new .env file in any text editor.

    cp .env.example .env
  4. 04

    Add the required credentials

    Paste a valid OpenAI API key into .env. Keep this file private and never commit it.

    OPENAI_API_KEY=your-openai-api-key OPENAI_MODEL=gpt-5.6 AGENT_API_KEY=change-me
    Credential note

    AGENT_API_KEY=change-me works for local judging. The OpenAI key must be supplied by the tester and is not included in the repository.

  5. 05

    Install and verify

    Install dependencies, run the automated tests, then check local readiness. The doctor may warn that change-me is the default local key; that key intentionally matches the plugin's default for this judge test.

    npm install npm run typecheck npm test npm run doctor
  6. 06

    Install the Studio plugin

    Run the included sync script, then fully restart Roblox Studio.

    chmod +x studio-plugin/sync.sh ./studio-plugin/sync.sh
  7. 07

    Start the backend

    Keep this Terminal window open while testing. A successful start listens on port 3100.

    npm start
  8. 08

    Connect Roblox Studio

    Open a place, enable Game Settings → Security → Allow HTTP Requests, then select Plugins → Roblox AI Agent → Agent.

    ✓ The panel should show a connected session.

03

Quick fixes

If something is not connected.

The plugin says offline

Confirm npm start is still running, HTTP requests are enabled in Studio, and the plugin was installed before Studio was restarted.

The API key is rejected

Confirm OPENAI_API_KEY has a valid key with model access. Do not add quotation marks around the value.

The doctor says not ready

Follow each failed line from npm run doctor. The Meshy key is optional; the OpenAI key, custom agent key, and installed plugin are the core checks.

A Studio change is waiting

Read the confirmation shown in the plugin and approve it within 20 seconds. Destructive actions intentionally fail closed when not approved.
